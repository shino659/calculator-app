# calculator-app

シンプルな静的 Web 電卓です。四則演算やパーセント、符号切り替え、キーボード入力に対応しています。ビルド不要で、そのままブラウザで開けます。

## 使い方
1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開くだけで利用できます（開発サーバー不要）。

## 操作方法
- ボタン入力: 画面上のボタンをクリックまたはタップして操作します。
- キーボード入力: 以下のキーに対応しています。
  - 数値: `0-9`
  - 演算子: `+`, `-`, `*`, `/`
  - 実行: `Enter` または `=`
  - 小数点: `.`
  - 取り消し: `Backspace`（1 桁削除）
  - 全クリア: `Escape`
- 追加機能: `%`（パーセント換算）、`±`（符号切り替え）、`C/AC`（入力値クリア / 全クリア）

## GitHub Pages での公開
静的ファイルのみで構成されているため、そのまま GitHub Pages で公開できます。`main` ブランチにプッシュし、リポジトリの GitHub Pages 設定で `main` ブランチのルートを公開対象にしてください。
