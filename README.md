# ブロックゲーム

キャンバスで遊べるシンプルなブロック崩しゲームです。矢印キーでパドルを操作し、ボールを落とさずにブロックをすべて壊しましょう。

## 特徴
- レベルが進むと行数が増えて難易度アップ
- 多段階耐久のブロックでスコアが変化
- スコア / ライフ / レベルがリアルタイムに表示
- キーボード操作: ← → で移動、スペースで開始・再開、P でポーズ
- ボタンでいつでもリスタート可能

## 遊び方
1. `index.html` をブラウザで開きます。
2. 画面下の「ゲームスタート / リスタート」ボタン、またはスペースキーでゲーム開始。
3. 矢印キー（← →）でパドルを動かし、ボールを反射させてブロックを壊します。
4. ボールを落とすとライフが減少。ライフがゼロになるとゲームオーバーです。
5. すべてのブロックを壊すと次のレベルへ進みます。レベル 5 をクリアすると完全勝利！

## 開発メモ
- 追加ライフや特殊ブロックなどの拡張もしやすい構成です。
- コードはプレーンな HTML/CSS/JS のみで動作します。
