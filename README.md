# calculator-app

シンプルな静的 Web 電卓です。四則演算やパーセント、符号切り替え、キーボード入力に対応しています。ビルド不要で、そのままブラウザで開けます。

## 使い方
1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開くだけで利用できます（開発サーバー不要）。

## 操作方法
- ボタン入力: 画面上のボタンをクリックまたはタップして操作します。
- キーボード入力: 以下のキーに対応しています。
  - 数値: `0-9`
  - 演算子: `+`, `-`, `*`, `/`
  - 実行: `Enter` または `=`
  - 小数点: `.`
  - 取り消し: `Backspace`（1 桁削除）
  - 全クリア: `Escape`
- 追加機能: `%`（パーセント換算）、`±`（符号切り替え）、`C/AC`（入力値クリア / 全クリア）

## 計算履歴（テープ）
- 表示欄の下に履歴パネルがあります。直近 10 件の「式 = 結果」が新しい順に並び、クリックまたは Enter/Space で結果を再入力できます。
- 「Clear history」ボタンで履歴を全削除できます。履歴は自動的に localStorage に保存され、ページを再読み込みしても残ります。

## コピー / ペースト
- 表示欄横の **Copy** ボタンで現在の表示値をクリップボードにコピーできます。失敗時はメッセージを表示します。
- Ctrl+V / Cmd+V の貼り付けに対応しています。数値・小数点・先頭のマイナスのみを受け付け、それ以外が含まれる場合は貼り付けを無視します。

## テーマ切替
- ヘッダー右の 🌙 / ☀️ ボタンでダーク / ライトテーマを切り替えられます。
- 初回は OS の設定に合わせて自動選択し、選択したテーマは localStorage に保存され次回以降復元されます。

## GitHub Pages での公開
静的ファイルのみで構成されているため、そのまま GitHub Pages で公開できます。`main` ブランチにプッシュし、リポジトリの GitHub Pages 設定で `main` ブランチのルートを公開対象にしてください。
